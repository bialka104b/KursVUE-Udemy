UDEMY

npm i -g @vue/cli  instalacja globalna vue
npm i -g @vue/cli-init dodatek
npm i -g vue

npm run vue init webpack uruchomienie projektu za pomocą webpack pod warunkiem że są zainstalowane paczki node_modules w tym projekcie i jest package.json a cały folder jest pusty

To get started:

  cd init
  npm run dev
